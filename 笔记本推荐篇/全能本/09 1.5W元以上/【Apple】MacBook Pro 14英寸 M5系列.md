## 优缺点

注意：M 处理器的 Mac 仅可运行 macOS 与部分 Linux 发行版，**无法运行 Windows**，这对于部分领域（比如机械制图、工业控制等，以及绝大部分网络游戏）是致命伤，但也是另一部分领域（比如平面设计、影视后期、数字音乐创作、软件开发等）的更优解乃至唯一解，因此不作为优缺点。

macOS 若需对接 Windows 生态，可使用第三方开发的兼容层或虚拟机（Wine/CrossOver、VMWare Fusion、UTM、Parallels 等）。这需要一定技术能力，对于多数常用软件及单机游戏无兼容性问题，但不能保证专业软件的性能与功能，也无法通过网游反作弊检测。

此外，对于习惯 Windows 操作逻辑的老手，迁移到 macOS 的学习成本较高；但对于几乎没接触过电脑的萌新，macOS 更接近移动端的交互设计反而更易上手。

**请读者务必确认自身的软硬件需求后，再行选购。**

### 优点

1. 能效优秀，相比超大 SoC 竞品，能效和性能全面领先；相比传统分立 CPU 与 GPU 的竞品，能效大幅领先且性能不落后
2. 续航持久，在离电不降性能的同时，处于续航时长的第一梯队（笔吧续航脚本成绩 13 小时以上）
3. 屏幕的亮度、色彩和对比度等素质极高，且表面抗眩光抗反射镀层或纳米玻璃纹理均为业内顶尖，观感极佳
4. 扬声器和麦克风极为优秀，是业内最好、没有之一
5. 集成双路视频编解码器，并支持 YUV444、ProRes、ProRes RAW 等专业格式的硬件加速
6. 睡眠可靠，合盖不掉电、开盖立即恢复，如同手机般无感，长期不关机也基本不会积攒小毛病
7. 做工优秀，质感极佳，市售第一梯队
8. 触控板为全域压感、玻璃疏油表面的多点触控板，手感优异，游戏场景之外是鼠标的上位替代（仅代表浪迹星海的个人体验与观点），市售最佳
9. 三雷电 5（Apple 称为雷雳 5）接口，快充支持 PD 3.1 EPR 140W，规格高且全能，且 Apple 设备的雷电兼容性是业内最佳
10. 日用安静且低温，几乎不会感觉到风扇起转、机身温度上升
11. 与 Apple 生态高度集成，和 iPhone、iPad、Apple Vision、Apple Watch、Apple TV、AirPods、HomePod 等产品之间有诸多跨平台联动与接力，系统与生态深度融入 iCloud 云服务
12. 3.5mm 音频口支持高阻抗耳机，声音处理延迟低
13. 铝合金一体成型的机身兼具高强度，基本无需担心日常的磕碰挤压跌落损伤
14. 品控较好，且官方售后网点较多
15. 基础款的性价比相对较高，非游戏场景下，同价位无可匹敌

### 缺点

1. 近期涨价，到手价提升了约六分之一，性价比降低
2. 巨大而意义不明的屏幕刘海设计，严重干扰观感与 macOS 顶部菜单栏显示，不提供人脸识别功能
3. 机身过于厚重，且边缘进气口割手，便携性较差
4. 性能释放过低，仅 50W 的性能释放对不起超过 1.5kg 的厚重机身
5. 高负载时键盘温度过高，铁板烤熊掌，噪音也很大
6. 内部没有扩展升级潜力：内存集成在处理器内，目前无可能升级；外存则集成在主板上，更换升级只能重新焊接闪存颗粒，破坏保修
7. 128GB 大内存款长期无货
8. macOS 的鼠标回报率默认很低，使用鼠标时指针略不跟手
9. 因为软件原因，雷电接口不支持外接独立 GPU
10. 缺乏常用的 USB Type-A 接口，若需要接驳则需另行购买拓展坞
11. 在越南与印度代工的批次，品控标准有所下降
12. ~~背面的苹果标不会亮，不利于星巴克装X~~

## 拓展性

内存与处理器封装在一起，不可加装或更换。

固态硬盘集成于主板，仅可通过重新焊接闪存颗粒的方式更换升级，破坏保修。

## 购买建议

1. 需要性能强大、续航持久且相对便携的移动工作站
2. 有极高的影音创作或观看要求
3. 不会用到不兼容 macOS 与虚拟机的软硬件

## 结语

关于 MacBook 的特点、与其他 Windows 笔记本的区别，MacBook Air 篇已经有所论述，下面说点特别的。

乔布斯回归 Apple 后，为了梳理当时杂乱的电脑产品线，定义了产品四象限：

|        | 固定式   | 便携式    |
| ------ | -------- | --------- |
| 消费级 | iMac     | iBook     |
| 专业级 | PowerMac | PowerBook |

其中，便携式分类的 iBook 和 PowerBook 就分别是后来 MacBook Air 与 MacBook Pro 的前身，Pro 在此指代其原意 Professional（专业的）而非指代简单的高配，意味着 MacBook Pro 的设计取向和 Air 大相径庭，特化面向特定的专业领域而非一般消费者，并不适合大多数人。

因此，MacBook Pro 的对手，不再是幻 14 Air、灵刃 14 等消费级轻薄游戏本，而是 ZBook Studio、ThinkPad P 系列等移动工作站；比较的也不再是游戏画质帧率，而是专业软硬件的兼容性、专业任务的性能、整机的可靠性等。此外，得益于 Mac 一贯的超高带宽的统一内存设计，以及 M5 系列 GPU 新引入的张量核心，MacBook Pro 是如今最适合人工智能大模型开发和应用的移动端平台，尤其在模型的参数规模达到生产可用的数百亿级别后。

如果读者只是希望凑齐苹果全家桶长面子，那么 MacBook Air 更加轻盈便携优雅，也要便宜得多；如果读者追求很好的游戏性能，那么 MacBook Pro 不是不能选，但性价比很低；如果读者需要调试单片机、绘制机械草图，那么 MacBook Pro 会让你痛不欲生砸电脑；但如果读者从事专业的音乐、摄影、影视、平面设计、软件开发等工作，那么 MacBook Pro 会让你得心应手甚至如获至宝。

彼之砒霜，吾之蜜糖。此等专业工作站的选择一定要切合自身需求，方能发挥出最大的价值。
